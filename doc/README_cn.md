# MacAIM - 自动输入法切换

MacAIM是一个轻量级工具，它会根据你正在使用的应用程序自动切换输入法。

每当你切换应用程序时，MacAIM 会自动调整输入法，旨在为你提供流畅高效的多语言输入体验。

请访问[发布页面](https://github.com/JimZhouZZY/MacAIM/releases)以获取最新的发布版本。
